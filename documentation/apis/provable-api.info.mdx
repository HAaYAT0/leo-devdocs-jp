Provable API は、ブロックレベルの情報やトランザクション、トランジションなどを照会できる RESTful エンドポイントを通じて Aleo ブロックチェーンへアクセスできるように設計されています。開発者はデータ取得とトランザクション送信の両方を通じて Aleo と直接連携できます。

## API ベース URL

| 環境 | URL |
|--------------|---------------------------------------------|
| Mainnet API  | https://api.explorer.provable.com/v1/mainnet |
| Testnet API  | https://api.explorer.provable.com/v1/testnet |
| Canary API   | https://api.explorer.provable.com/v1/canary  |

## レート制限

ユーザーは **1 秒あたり最大 5 リクエスト**、**1 日あたり最大 100,000 リクエスト** まで実行できます。

レート制限は厳格に適用され、API を濫用した場合はブラックリスト対象となります。より高いレート制限については devservices@provable.com までお問い合わせください。
