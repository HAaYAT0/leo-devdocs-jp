```bash title=ENDPOINT
GET /{network}/transaction/{ID}
```

指定したトランザクション ID に対応するトランザクションを返します。

### 引数

| パラメーター       |  型  | 必須 | 説明                                         |
|:----------------|:------:|:--------:|:----------------------------------------------------|
| `ID` | string |   Yes    | 取得したいトランザクションのトランザクション ID（16 進表現） |

### レスポンス

| パラメーター |                    型                     |        説明        |
|:---------:|:-------------------------------------------:|:-------------------------:|
| `result`  | [object](../concepts/fundamentals/03_transactions.md) | 要求されたトランザクション |
