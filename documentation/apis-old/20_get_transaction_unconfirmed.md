```bash title=ENDPOINT
GET /{network}/transaction/unconfirmed/{ID}
```

指定したトランザクション ID の未確定トランザクションの詳細を取得します。

### 引数

| パラメーター |  型  | 必須 | 説明                                         |
|:----------|:------:|:--------:|:----------------------------------------------------|
| `ID`      | string |   Yes    | 取得したいトランザクションのトランザクション ID（16 進表現） |

### レスポンス

| パラメーター |                    型                     |        説明        |
|:---------:|:-------------------------------------------:|:-------------------------:|
| `result`  | object | 要求された未確定トランザクション |
