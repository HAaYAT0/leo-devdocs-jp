```bash title=ENDPOINT
POST /{network}/transaction/broadcast
```

トランザクションを台帳にブロードキャストします。

### ボディ

|  型  |               説明               |
|:------:|:---------------------------------------:|
| string | ブロードキャストするシリアライズ済みトランザクション |

### レスポンス

| パラメーター |  型  |                   説明                   |
|:---------:|:------:|:-----------------------------------------------:|
| `result`  | string | トランザクションのブロードキャスト試行のステータス |
