```bash title=ENDPOINT
GET /{network}/find/blockHash/{transactionID}
```

指定したトランザクション ID を含むブロックのブロックハッシュを返します。

### 引数

|    パラメーター    |  型  |    説明     |
|:---------------:|:------:|:------------------:|
| `transactionID` | String | トランザクション ID |

### レスポンス

| パラメーター |  型  |                 説明                  |
|:---------:|:------:|:--------------------------------------------:|
| `result`  | String | トランザクション ID を含むブロックのブロックハッシュ |
