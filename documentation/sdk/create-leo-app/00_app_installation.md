## 最初の Aleo プロジェクトをスキャフォールドする

:::note
**互換性:**
[Node.js](https://nodejs.org/en/) バージョン 18 以上
:::

NPM を使う場合:

```bash
npm create leo-app@latest
```

1. プロジェクト名を入力します。
2. 次の選択肢から好みのフレームワークを選びます。
    - `React`
    - `Node.js`
    - `Vanilla (JavaScript)`

3. フレームワークに `React` を選択した場合、利用できるテンプレートは以下のとおりです。
    - `JavaScript` + `Leo`
    - `TypeScript` + `Leo`
    - `TypeScript` + `Next.js`
